About

Some background info on this project and its author...

Project

This is a personal project I built to gain experience in web development and ML deployment. The data is downloaded from Aleksey Korshuk's HuggingArtists. The models are GPT-2 architectures that I fine-tuned on artist-specific datasets. The webapp is built using Flask, a Bootstrap template, and deployed on Heroku.

You can find the code on my GitHub repo. Please give the correct attribution if reusing any part of it.

Author

I'm Luc Frachon, a Machine Learning scientist with a PhD from Heriot-Watt University (UK). I have extensive academic, startup, and industrial experience in Data Science, ML, and Deep Learning. I am always hungry to learn new skills (hence this website).
